# Categories

Categories model and traits for MyProject.

## Installation

```bash
composer require datasdk/categories
```

## Autoload

```json
{
    "autoload": {
        "psr-4": {
            "MyProject\\Categories\\": "src/"
        }
    }
}
```

## Namespace

- Model: `MyProject\Categories\Models\Categories`
- Trait: `MyProject\Categories\Traits\Categories`
